# Моя ~~крутая~~ практическая

## *Описание проекта*

Простое **консольное приложение** для управления задачами на Python.  
Программа позволяет работать со списком дел через терминал.

### Основной функционал:
- Добавление новых задач
- Просмотр списка задач
- Отметка задач как выполненных
- Удаление задач
